# K44DDK AI Platform

## Mục tiêu

Xây dựng nền tảng AI dành riêng cho cộng đồng K44DDK.

## Chức năng

- AI hỏi đáp
- Chat cộng đồng
- Tra cứu cựu sinh viên
- Tìm kiếm bài viết
- Đồng bộ Blogger
- Đồng bộ Excel
- Quản trị dữ liệu

## Công nghệ

Frontend:
- JavaScript
- HTML
- CSS

Backend:
- Firebase
- Google Apps Script

AI:
- OpenAI API

Nguồn dữ liệu:
- Blogger
- Google Sheets
- Excel
